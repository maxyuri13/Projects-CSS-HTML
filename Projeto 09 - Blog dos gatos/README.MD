### 👀 Métodologias utilizadas 
  - Responsividade (Mobile/Desktop)
  - Unidade de medida flexível
  - Grid
  - Variáveis no CSS
  - Cores HSL
  - Animações e transições simples

### ✈️ Tecnologias utilizadas
  - HTML
  - CSS
  - Figma
  - VSCode (Editor de código)
